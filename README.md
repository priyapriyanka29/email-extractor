# 📧 Email Extractor

## 📌 Overview

A Python tool that extracts email addresses from text files using pattern matching.

## 🚀 Features
                        
* Extract emails from text
* Uses regex for pattern detection

## 🛠️ Tech Used 

* Python
* Regular Expressions

## ▶️ Run

```bash
python main.py
```

## ⚠️ Note

Works best with structured text data.
